# STOCKFLOW

Sistema de gerenciamento de estoque desenvolvido em Java utilizando conceitos de Programação Orientada a Objetos, SOLID e arquitetura em camadas.

---

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Remoção de quantidade do estoque
- Remoção automática quando estoque zera
- Menu interativo no terminal

---

## Tecnologias Utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub

---
